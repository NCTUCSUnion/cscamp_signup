## 圖片格式與尺寸建議

- Banner 圖片：建議寬度 1920px，高度 400-500px
- 幻燈片圖片：建議寬度 1200px，高度 800px
- 團隊照片：建議寬度 800px，高度 600px
- 卡片圖片：建議寬度 600px，高度 400px

請確保所有圖片均為高品質且已壓縮，以提升網站加載速度。
